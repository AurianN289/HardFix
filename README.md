# HardFix
Projeto destinado a uma ideia de comunidade de resolução de problemas de hardware


A estrutura do projeto vai ser a seguinte

front end - back end : cada um responsável por uma parte da aplicação

dentro do front: 
    assets: aqui vão as imagens, audios, videos, e quaisquer outras midias que o projeto use
    components: aqui vão componentes reutilizáveis durante o projeto: botoes, navbar, footer, etc
    pages: aqui vão as paginas do projeto: home, login, cadastro, etc
    service: aqui vão solicitações de acesso ao back end, uma forma de separar quem se comunica com back end no front: gets e posts para o back end
